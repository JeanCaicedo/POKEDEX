# POKEDEX

Ejercicio de primeros pasos con JavaScript vanilla: una Pokédex que consume la PokéAPI pública para listar y filtrar Pokémon por tipo.

## Contexto

Práctica de autoestudio para reforzar HTML, CSS y JavaScript vanilla consumiendo una API REST pública. El ejercicio se centra en `fetch`, manipulación del DOM y filtrado por tipos de Pokémon.

## Tecnologías

- HTML5
- CSS3
- JavaScript (vanilla)
- [PokéAPI](https://pokeapi.co/)

## Ejecución

```bash
# Abrir index.html directamente en el navegador
# o servirlo con cualquier servidor estático, por ejemplo:
npx serve .
```
